# Django-login
## A builerplate code to start a django project with User Authentication
```
git clone https://github.com/vishal-pandey/django-login.git
cd django-login
virtualenv env -p python3
source env/bin/activate
pip install -r requirements.txt
open http://127.0.0.1:8000/
```
